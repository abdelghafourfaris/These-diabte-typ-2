# These-diabte-typ-2
Le diabète de type 2 est une maladie chronique caractérisée par une résistance à l’insuline et une élévation prolongée de la glycémie. Mon projet vise à identifier de nouveaux inhibiteurs naturels capables d’agir sur plusieurs protéines impliquées dans la régulation du glucose. Grâce à des techniques in silico telles que le docking moléculaire.
